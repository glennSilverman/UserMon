UserMon by glennSilverman

Example Lift application using ManyToMany trait on Mapper

Demonstrates adding role-based user login and administration in a Lift application. Uses new ManyToMany trait to manage user roles and JQuery asmselect javascript for editing.

Dependencies

asmselect javascript and mysql jdbc driver. Included in repository.

Install

You can use derby for your database if you must, but the program is configured for mysql, so you will need to have a running instance of mysql. The pom includes the mysql jdbc driver as a dependency. Create a database schema (the default is lift_usermon) to store the Lift mapper entities and configure for your mysql username and password, like so, val dbUrl: String = Props.get("db.url") openOr "jdbc:mysql://localhost:3306/lift_usermon?user=&password=" in Boot.scala.
